Activation

Cocher cette case pour commencer à utiliser l'extension sur le blog courant.

Note : la désactivation ne supprime pas les liens entre les billets, elle désactive simplement leur affichage en partie publique et dans la page d'édition des billets.

Ajouter des liens entre les billets

Pour ajouter des liens à un billet, éditer le billet et cliquer sur Billets connexes : sous le champ Notes :

Options d'affichage

Titre du bloc

Le titre sera affiché avant le bloc contenant les liens. Laisser vide pour ne pas afficher de titre.

Position du bloc

Le bloc sera affiché au-dessus ou au-dessous du contenu des billets. Pour n'utiliser que le widget, décocher les deux cases.

Options d'extraction des images

Note : toutes les options suivantes nécessitent d'installer l'extension listImages. Les mêmes options d'extraction seront alors aussi disponibles dans le widget de présentation.

Extraire les images des billets connexes

Si cette case est cochée, les liens vers les billets connexes seront affichés sous forme d'images extraites des billets. Sinon, les liens seront affichés sous forme d'une simple liste.

Origine des images

Les images peuvent provenir de l'extrait du billet, de son contenu, ou du billet complet.

Taille de l'image

Choisir dans la liste la taille des images affichées. Si le format choisi n'est pas présent dans le gestionnaire de médias, l'image sera ignorée.

Inclure les dimensions des images

Cocher cette case pour ajouter les attributs width et height à la balise img. Cette option ne conviendra pas si votre thème utilise des mediaqueries.

Attribut alt des images

Choisir ou non d'ajouter un attribut alt à la balise img.

Première image à extraire

Choisir ici le numéro d'ordre de la première image extraite du billet. Si cette valeur est supérieure au nombre total d'images dans le billet, aucune image ne sera extraite.

Nombre d'images à extraire

Choisir ici le nombre d'images à extraire pour chaque billet. Si cette valeur est supérieure au nombre total d'images dans le billet, toutes les images sont extraites. Si une image ne possède pas de miniature au format demandé elle ne sera pas décomptée.

Légende

Affiche ou non une légende sous l'image. Cette légende peut contenir le titre du billet ou celui de l'image originale.

Balise HTML

Choisir la balise HTML englobant les images. Valeurs possibles : li (élément de liste), div (boîte), none (aucune balise).

Un attribut CSS "portrait" ou "landscape" sera ajouté à cette balise en fonction du sens de l'image.

Classe CSS

Entrer ici une valeur pour ajouter un attribut class à la balise img.

Destination des liens

Les liens peuvent pointer vers le billet ou vers l'image originale. Il peut aussi ne pas y avoir de liens.

Infobulle

Affiche ou non une infobulle au passage de la souris sur l'image. Cette infobulle peut contenir le titre du billet ou celui de l'image originale.

Filtres de la liste des billets

Auteur
Permet de filtrer les billets par auteur.
Catégorie
Filtrer les billets par catégorie.
État
  • en attente : en attente de publication.
  • programmé : le billet sera mis en ligne aux date et heure indiquées dans le champ Publié le.
  • non publié : billet hors ligne.
  • publié : billet en ligne.
Sélectionné
Aucun, billet marqué comme sélectionné ou non sélectionné.
Mois
Filtre les billets d'un mois d'une année donné.
Langue
Filtre les billets selon la langue indiquée dans le champ langue du billet.
Trier par
Permet de trier les résultats de filtrage selon la date, le titre, la catégorie, l'auteur, l'état de publication ou l'état de sélection.
Ordre
Indique l'ordre dans lequel on souhaite effectuer le tri.
Billets par page
Nombre de billets à afficher par page de résultat.

Action par lot sur les billets

Il est possible de supprimer la totalité des liens pour un ou plusieurs billets.

Si vous souhaitez ne supprimer qu'un seul lien d'un billet à un autre, allez dans la page d'édition de l'un des deux billets.
