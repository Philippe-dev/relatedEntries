Activation

Check this box to start using related posts on this blog.

Note : deactivation does not delete links between posts, it merely deactivates their display on public pages and in the posts editing page.

Display options

Block title

Title will be displayed before links block. Leave empty to display no title.

Block position

Block will be deisplayed before or after post content. Uncheck both boxes to use only the widget.

Images extracting options

Note : all following options require listImages plugin. Same options will be then available in the presentation widget as well.

Extract images from related posts

If this is checked, links to related entries will be displayed as images extracted from posts. Otherwise, links will be displayed as a simple list.

Images origin

Images can be extracted from post abstract, post content, or both.

Image size

Choose the image size from list. If the selected size does not exist, image will be ignored.

Include images dimensions

Check to add width and height attributes to img tag. This option should not be used if your theme uses media queries.

Images alt attribute

Choose to add or not alt attribute to img tag.

First image to extract

Choose the first image to be extracted. If this value is bigger than the post's total amount of images, no image will be extracted.

Number of images to extract

Choose here how many images will be extracted from each post. If this value is bigger than the post's total number of images, all images will be extracted. If an image does not have a requested size, it will not be counted in the number of images.

Legend

Displays a legend under the image. This legend may contain post title or image title.

HTML Tag

Choose the HTML tag including images. Possible values : li (list element), div (box), none (no tag).

CSS attributes "portrait" or "landscape" will be added to this tag as required by image orientation.

CSS class

Type here a value to add an extra CSS class attribute to img tag.

Links destination

Links can lead to original entry or real size image. Links can be also removed.

Bubble

Displays a bubble over links when hovered by pointer. This bubble may contain post title or image title.

Filters for the entry list

Author
Allows you to filter by author.
Category
Filter by category.
Status
  • pending: posts whose publication status has not yet been decided.
  • scheduled: post to be set online automatically.
  • unpublished: offline posts.
  • published: online posts.
Selected
Only the selected posts, or the non selected ones.
Month
Filter on a specific month and year.
Language
Filter on the different languages used on the blog.
Sort by
Defines the list sorting parameter, either on the date, the title, the author, the status or the selected status.
Order
Choose whether the list will be sorted in ascending or descending order.
Entries per page
Number of entries that will be displayed on each page of the resulting set.

Selected entry actions

It is possible to delete all links for one or several related posts.

However, if you wish to delete only one link from one entry to another, go to one of the entries editing page.
